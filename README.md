# StudyFollow

## Overview
StudyFollow is a student web app for managing academic tasks, upcoming exams, and study organization in one place.

## The Problem This Project Solves
Many students manage tasks, exams, and study planning in a scattered way across notes, messages, and general-purpose apps.  
StudyFollow brings everything into one clear and organized system.

## Target Audience
Students, especially higher-education students, who need a simple and convenient way to track tasks and exams.

## Competitors and Differentiation
Current alternatives include:
- Excel / Google Sheets
- Notes apps
- WhatsApp messages to self
- Digital calendars
- Manual planning

### Differentiation
- Built specifically for academic management
- Combines tasks, exams, and personal profile in one place
- Clean, responsive, RTL-friendly interface
- Live data persistence using Supabase
