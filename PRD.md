# STUDYFOLLOW PRD

## Product
STUDYFOLLOW is a student productivity web app that helps students manage classes, tasks, exams, focus sessions, and profile settings in one visually organized dashboard.

## Core user problem
Students often split their academic planning across notes, WhatsApp groups, calendars, and memory. This creates stress, missed deadlines, and poor prioritization.

## Solution
Provide a single dashboard with quick visibility into today's classes, pending tasks, upcoming exams, and reminders.

## Core pages
- Login / Register
- Dashboard
- Tasks
- Exams
- Profile

## Stage scope
Frontend only. Dummy data only. No real backend.
